



# 申论概述

## 申论答案
### 得分点对照

| **命题人侧**                         | **阅卷员侧**                                                        |
|----------------------------------------|-----------------------------------------------------------------------|
| 1. 结合农村实际                        | 1. 结合农村实际（或“乡村版”，只写“垃圾分类法”不可以）                   |
| 2. 做好组织保障                        | 2. 做好组织保障（成立项目实施小组）                                    |
| 3. 工作有序                            | 3. 工作有序（或制定路线图，有计划、有步骤）                            |
| 4. 做好财政支持                        | 4. 对人财物的支持始终如一（只写人财物支持不可以）                        |
| 5. 处理流程科学合理                    | 5. 科学合理的垃圾处理流程（或常态化、制度化，或投放收集环环相扣）        |
| 6. 收费合理                            | 6. 收费合理（或制定收费标准，聘请垃圾清运员）                          |
| 7. 赏罚分明                            | 7. 制定规则，奖罚分明                                                 |
| 8. 注重社会宣传，和理念培养            | 8. 社会宣传、理念培养（或宣传教育培训及时跟进、为环保理念造势）          |
| 9. 从孩子（学生）抓起                  | 9. 对学生教育引导（或学科老师在课堂引导孩子）                          |
| 10. 商机经济效益                       | 10. 商机经济效益（捕捉商机，获得经济效益）                              |



1. **结合农村实际**：指在制定和实施政策时，需充分考虑农村的具体情况和特点，确保政策的可行性和针对性。例如，农村地区可能面临基础设施薄弱、居民教育水平较低等挑战，因此在推广垃圾分类时，需因地制宜，采用适合当地的方式。

2. **做好组织保障**：指在政策实施过程中，要有专门的组织和人员负责协调和执行，确保各项工作顺利推进。例如，可以成立项目实施小组，明确职责分工，保证政策的落地和持续推进。

3. **工作有序**：指在政策实施中，需有清晰的计划和步骤，避免混乱和随意。例如，制定详细的工作路线图，确保各项工作按部就班地推进，并有可供调整的预案。

4. **做好财政支持**：指政策的实施离不开充足的资金支持，需要确保财政投入的连续性和有效性。财政支持应覆盖人力、物力和财力各个方面，确保资源的合理配置和使用。

5. **处理流程科学合理**：指在政策实施中，需设计科学合理的流程，确保每一个环节紧密衔接。例如，在垃圾分类过程中，投放、收集、运输、处理等环节需要无缝衔接，确保处理流程的高效性和可持续性。

6. **收费合理**：指在提供公共服务时，收费标准应合理制定，既能覆盖成本，又不会给居民带来过大的经济负担。例如，可以通过聘请专业的垃圾清运员，确保服务质量，并根据当地经济水平设定合理的收费标准。

7. **赏罚分明**：指在政策执行过程中，需制定明确的规则，对表现好的个人或集体进行奖励，对违反规定者进行处罚，以确保政策的有效执行。

8. **注重社会宣传，和理念培养**：指在政策推广中，需加强宣传教育，提升公众的环保意识。例如，可以通过各种媒体宣传环保理念，组织教育培训活动，营造良好的社会氛围。

9. **从孩子（学生）抓起**：指环保教育应从学生抓起，通过学校教育培养孩子的环保意识，使其成为环保的践行者和宣传者。

10. **商机经济效益**：指在政策实施中，可以捕捉到潜在的商机，产生经济效益。例如，通过推广垃圾分类，可以发展循环经济，推动环保产业的发展，实现经济和环境效益的双赢。

### 解答要点
### 答案给谁看？什么是好答案？

- **答案归根到底都是写给阅卷员看的**  
  跟标答一样的答案当然最好，但是我写不出来。

- **满足阅卷员的需求**  
  答案应该是满足阅卷员的需求，而不是满足所谓领导、政府部门的需求。

- **“快申论”思路**  
  在考场上能够快速写出来且能够快速得分的就是最好的答案。  
  因此，这种思路被称为“快申论”。

- **以“能够得分”为最终目的**  
  所以答案以“能够得分”为最终目的。

### 树立考试思维

1. **所有答案源于材料**  
   对材料进行分析、查找，节约时间，找出所有答案要点。

2. **不追求“好”、“美”、“高大上”的答案**  
   我们只写自己能做到的且能得分的答案。

3. **基于材料提炼**  
   如果个人能够提炼，必须基于材料提炼，只要能得分，就是“最好”的答案。

4. **抄原文照样得分**  
   如果提炼能力不足，就抄原文，照样得分。

## 阅卷规则
#### 小题：关键词（客观标准）

- **材料原词优先得分**  
  答题时应尽量使用材料中的原词、原句，因为80%以上的得分来自于直接引用材料中的词语。剩下的20%左右需要考生对材料进行理解、总结后用自己的语言表达。**在公务员考试的申论小题中，得分的关键在于对材料中关键词的把握。这意味着考生需要非常仔细地阅读材料，确保提取到的内容直接来自于材料文本。**

- **固定的词语得分**  
  在小题部分，只有特定的固定词语可以得分，其他词语即使表达了类似的意思，也可能不得分。**这强调了在答题时，对材料原词的准确引用和使用的重要性。**

#### 作文：整体观照、分等级赋分

- **整体观照**  
  作文题目采用整体观照、分等级赋分的方式，而不是像小题一样的关键词评分。因此，作文是真正的主观题，考生需要注重文章的整体结构和逻辑性。**作文的结构、逻辑、论证等整体因素比单个关键词更加重要。**

- **分数区间**  
  阅卷老师在给作文打分时，并不需要给出非常准确的分数，而是根据文章的整体表现，将其划分到合理的分数区间内。这个区间段是宽泛的，评分更具弹性。**合理分数的概念也提醒考生，不必追求完美，只要达到一定的标准即可获得较好的成绩。**

## 题型
| **题型**         | **关键词**                                                                                      | **说明**                                                                                                                                                    |
|------------------|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **概括题**       | 概括、归纳、提炼、总结；分析·····原因、指出······问题/表现；特点、难点、要点；启示、教训、经验、学习、借鉴；标题、填空等。 | 概括题是所有题型的基础，要求考生对材料中的信息进行提炼和总结。                                                                                             |
| **分析理解题**   | 理解；认识；阐释；看待；看法；解释；含义；评析等。                                                | 分析理解题考察考生对材料的深层次理解和分析能力，要求对材料中的观点进行解释和评析。                                                                         |
| **对策题**       | 提出······对策（建议、措施、做法、意见等与对策同意替换的词）。                                    | 对策题要求考生针对材料中提出的问题，提出相应的解决方案或建议。                                                                                             |
| **公文题**       | 文体（如宣传稿、一封信、倡议书、通知、短文、短评、演讲稿、讲解稿、讲话稿等字样）。                | 公文题要求考生撰写特定文体的文章，字数一般在400字以上、800字以下。                                                                                         |
| **作文题**       | 议论文。                                                                                        | 作文题通常是最后一道题，要求考生写一篇议论文，考察其论证能力和文字表达能力。                                                                               |
| **应用文写作**   | 新类型写作。                                                                                    | 应用文写作是近年出现的新题型，可以理解为“篇幅在800字以上的公文题”，兼具公文题的格式和作文题的表达，要求考生在格式和表达上都要符合要求。 |
